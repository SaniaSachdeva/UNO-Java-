# UNO-Java-
This project is a Java-based implementation of the classic UNO card game. It simulates real-time gameplay using object-oriented programming principles and follows a structured software development lifecycle (SDLC) approach. The system handles core UNO rules, player turns, and card interactions in a modular and scalable design.

Tech Stack
Java
Object-Oriented Programming (OOP)
UML (Class Diagrams and Design)
Software Development Life Cycle (SDLC)

Features
Complete UNO gameplay logic implementation
Turn-based player system
Rule-based card actions such as Skip, Reverse, and Draw Two
Modular class structure for scalability and maintainability
Tested and debugged for consistent performance

Concepts Applied
Object-Oriented Programming including encapsulation, inheritance, and polymorphism
Data structures for managing decks and players
Software design using UML diagrams
SDLC practices including design, development, and testing

Future Improvements
Add graphical user interface using JavaFX or Swing
Implement multiplayer functionality
Introduce AI-based opponent logic
